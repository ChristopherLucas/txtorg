Windows installation

To install txtorg on a Windows machine, follow the steps below.

  1. Install Python (ideally 2.7 but definitely not 3.x)
  2. Install setuptools: Download https://bitbucket.org/pypa/setuptools/raw/bootstrap/ez_setup.py and double-click it! This will install easy_install.exe, which you'll need in step 5.
  3. Install Java (if it's not installed)
  4. Locate the jvm.dll file (mine is in C:\Program Files (x86)\Java\jre1.6.0_24\bin\client). You'll need this in the next step.
  5. Add the dll location and the easy_install location to the Path (easy_install.exe is located for me in C:\Python27\ArcGIS10.2\Scripts). If you haven't done this before, follow these steps.
  6. , right click on My Computer, click Properties.
  7. * Advanced System Settings => Environment Variables
  8. * Click PATH in the list of System Variables
  9. * Click edit, and change the path by adding, e.g.
  10. ;C:\Program Files (x86)\Java\jre1.6.0_24\bin\client;C:\Program Files (x86)\Java\jre1.6.0_24\bin\client
  11. * (ensure each location is separated by semicolons)
  12. Download this file and double click it
  13. Download the zip file in the left toolbar and double-click it.
  14. Open the directory where you unzipped the text-organizer, go to bin, rename txtorg to txtorg.py, double click it.
