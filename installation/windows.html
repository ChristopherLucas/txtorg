Windows installation

To install txtorg on a Windows machine, follow the steps below. We hope to update soon with more detailed instructions.

  1. Install Python (ideally, 2.7)
  2. Install setuptools: Download https://bitbucket.org/pypa/setuptools/raw/bootstrap/ez_setup.py and double-click it!
  3. Install Java (if it's not installed)
  4. Locate the jvm.dll file (mine is in C:\Program Files (x86)\Java\jre1.6.0_24\bin\client)
  5. Add the dll location and the easy_install location to the Path (easy_install.exe is located for me in C:\Python27\ArcGIS10.2\Scripts)
  6. To add to path, right click on My Computer, click Properties.
    * Advanced System Settings => Environment Variables
    * Click PATH in the list of System Variables
    * Click edit, and change the path by adding, e.g.
    ;C:\Program Files (x86)\Java\jre1.6.0_24\bin\client;C:\Program Files (x86)\Java\jre1.6.0_24\bin\client
    * (ensure each location is separated by semicolons)
  7. Download this file and double click it
  8. Download the zip file in the left toolbar and double-click it.
  9. Open the directory where you unzipped the text-organizer, go to bin, rename txtorg to txtorg.py, double click it.
