Linux installation

If you are using a Linux distribution, the installation process is fairly straightforward (isn't this why we're using Linux, after all?). First, download PyLucene 3.6.2 here (any version 3 release will do). From a shell, navigate to the directory in which you saved the tarball. Then, install openjdk, Apache Ant, the gnu c++ compiler, python dev, and PyLucene. Then, download and install txtorg as you would any other Python package. These steps are shown in the code block below for Debian-like distros. If your distro uses the RPM package manager, we trust that you know this and know what you need to change in the instructions below. If not, email Christopher. Also note that if you're using a 32-bit machine, you must edit the setup.py file such that it points to the right directory in which you're JDK is installed (the default path includes a folder containing the processor and assumes 64-bit).

$ cd this_is_where/I_saved_my_tarball
      $ tar -zxvf pylucene-[PyLucene Version Here]-src.tar.gz
      $ sudo apt-get install openjdk-7-jdk
      $ sudo apt-get install ant
      $ sudo apt-get install g++
      $ sudo apt-get install python-dev
      $ cd pylucene-4.5.1-1/jcc
      $ sudo python setup.py build
      $ sudo python setup.py build
      $ cd ..

Next, you need to edit the Makefile by uncommenting the section of code shown in the block below.

# Linux (Ubuntu 11.10 64-bit, Python 2.7.2, OpenJDK 1.7, setuptools 0.6.16)
      # Be sure to also set JDK['linux2'] in jcc's setup.py to the JAVA_HOME value
      # used below for ANT (and rebuild jcc after changing it).
      PREFIX_PYTHON=/usr
      ANT=JAVA_HOME=/usr/lib/jvm/java-7-openjdk-i386 /usr/bin/ant
      PYTHON=$(PREFIX_PYTHON)/bin/python
      JCC=$(PYTHON) -m jcc --shared
      NUM_FILES=4

Back in the terminal, finish the installation process by issuing the following commands.

$ make
      $ sudo make install
      $ cd /where/I/saved/the/txtorg/tarball
      $ tar -zxvf ChristopherLucas-txtorg-303111d.tar.gz
      $ cd ChristopherLucas-txtorg-303111d
      $ sudo python setup.py build
      $ sudo python setup.py install
